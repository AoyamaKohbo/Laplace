Laplace(ラプラス)変換と伝達関数

§Laplace変換

有限関数\(f(t)\)のLaplace変換は次式となります。

\begin{align} F(s) =\int_0^\infty f(t) e^{-st}dt\label{eq:LaplaceTrns} \end{align}

Laplace変換には次の公式があります。

\begin{align} \int_0^\infty 1 e^{-st}dt = \frac{-1}{s}[e^{-st}]_0^\infty = \frac{1}{s}\label{eq:LaplaceUnit} \end{align} \begin{align} \int_0^\infty \frac{df(t)}{dt} e^{-st}dt = [f(t)e^{-st}]_0^\infty+s\int_0^\infty \frac{df(t)}{dt} e^{-st}dt = sF(s) - f(0)\label{eq:LaplaceDiff} \end{align} \begin{align} \int_0^\infty f(t-T) e^{-st}dt = \int_0^\infty f(\tau) e^{-s(\tau+T)}dt = e^{-sT}\int_0^\infty f(\tau) e^{-s\tau}d\tau = e^{-sT}F(s)\label{eq:LaplaceDly} \end{align} \begin{align} \int_0^\infty e^{-at} e^{-st}dt = \int_0^\infty e^{-(s+a)t}dt = \frac{-1}{s+a}[e^{-(s+a)t}]_0^\infty = \frac{1}{s+a}\label{eq:LaplaceExp} \end{align}

§伝達関数

入力信号\(u(t)\)に対する出力信号\(y(t)\)の関係を次式とします。 ただし、\(d^ku(t)/dt^k|_{t=0} =0~(k=0,\cdots,m-1)\)、\(d^ky(t)/dt^k|_{t=0} = 0~(k=0,\cdots,n-1)\)とします。

\begin{align} \sum_{k=0}^n b_k\frac{d^k y(t)}{dt^k} = \sum_{k=0}^m a_k\frac{d^k u(t)}{dt^k}\label{eq:DgtFlt} \end{align}

\(u(t)\)と\(y(t)\)のラプラス変換をそれぞれ\(U(s)\)と\(Y(s)\)とし、 式\eqref{eq:DgtFlt}をラプラス変換すると公式\eqref{eq:LaplaceDiff}により次式となります。

\begin{align} \sum_{k=0}^n b_k s^k Y(s) = \sum_{k=0}^m a_k s^k U(s)\label{eq:LapDgtFlt} \end{align}

式\eqref{eq:DgtFlt}の伝達関数\(G(s)\)は次式となります。

\begin{align} G(s) = \frac{Y(s)}{U(s)} = \frac{\displaystyle\sum_{k=0}^n b_ks^k Y(s)}{\displaystyle\sum_{k=0}^m a_ks^k U(s)} = \frac{b_ns^n+\cdots + b_0}{a_ms^m+\cdots+a_0}\label{eq:TrnsFunc} \end{align}
